# RideSharing DApp (Ethereum Sepolia)
Final Exam Advanced Blockchain Programming

## Teknologi
- Solidity (Smart Contract)
- Ethereum Sepolia Testnet
- Remix IDE
- MetaMask
- Infura RPC
- Web3.js
- HTML (Frontend)

## Cara Menjalankan
1. Install MetaMask dan pilih jaringan Sepolia
2. Akun wallet harus mempunyai saldo SepoliaETH
3. Buka file 'index.html` dengan Live Server
4. Connect MetaMask
5. Simulasi sudah bisa dilakukan
